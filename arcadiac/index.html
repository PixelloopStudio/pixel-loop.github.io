Arcadiac by Pixelloop

Arcadiac is an arcade style fun game by Pixelloop.


Your goal is to bring as many objects in the scene as possible. When the screen is full of objects you lose the game. Watch out! You should shoot the object before the arrow gets red! Objects with the same shape and same color will delete each other, so you get some free space in the game!
